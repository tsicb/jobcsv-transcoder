# 求人CSV《コード⇔日本語》変換ツール

GitHub Pagesで公開できる静的ブラウザアプリです。CSVをアップロードすると、選択した列を「コード→日本語」または「日本語→コード」に変換し、自動でCSVをダウンロードします。

## 使い方

1. 変換モードを選びます。
2. 出力モードを選びます。
3. 必要に応じて「変換列を選ぶ」から対象列を変更します。
4. CSVをドラッグ＆ドロップ、またはクリックして選択します。
5. 選択後、自動で変換済CSVのダウンロードが始まります。

## 仕様メモ

- 対象外列はそのまま出力します。
- デコード時は `<BR>` を改行に変換します。
- エンコード時は改行を `<BR>` に変換します。
- 複数値列は `::` で分割し、個別に変換して `::` で再結合します。
- 未変換値は元値を残し、画面上に注意一覧を表示します。
- 市区町村は県コード列と市区町村コード列をセットで処理します。
- 駅名エンコードは、完全一致または一意一致のみ自動変換します。
- 入力文字コードは自動判定し、出力文字コードは原則として入力に寄せます。
